Context.ContextSigners Class

A class used to list and modify the signers of a context.

Syntax

public class Context.ContextSigners : IEnumerable<Key>

Remarks

The Context.ContextSigners class allows you to list (enumerate), add and remove (clear) all signers.

Add Method

Add a key to the list of signers.

Syntax

Parameters

The key that shall be used for cryptographic signing operations.

Remarks

After adding the key signer to the list of signers it will be used within the Context.Sign(GpgmeData, GpgmeData, SignatureMode) and the Context.EncryptAndSign(Key[], EncryptFlags, GpgmeData, GpgmeData) method.
