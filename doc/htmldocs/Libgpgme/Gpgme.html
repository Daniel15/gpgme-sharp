EngineCheckVersion Method

public static bool EngineCheckVersion (Protocol proto)

The function verifys that the engine implementing the protocol proto is installed in the expected path and meets the version requirement of GPGME.

Parameters

proto
A Libgpgme.Protocol that shall be verified.

Returns

Value Description
true The engine is available.
false The engine is not available.

Remarks

GPGME supports several cryptographic protocols, however, it does not implement them. Rather it uses backends (also called engines) which implement the protocol. GPGME uses inter-process communication to pass data back and forth between the application and the backend, but the details of the communication protocol and invocation of the backend is completely hidden by the interface. All complexity is handled by GPGME. Where an exchange of information between the application and the backend is necessary, GPGME provides the necessary callback function hooks and further interfaces.

GetHashAlgoName Method

public static string GetHashAlgoName (HashAlgorithm algo)

The function returns a string containing a description a hash algorithm.

Parameters

algo
A Libgpgme.HashAlgorithm you want the pretty name from.

Returns

The function returns a string containing a description of the hash algorithm algo. If algo is not a valid hash algorithm, null is returned.

Remarks

This string can be used to output the name of the hash algorithm to the user. Hash (message digest) algorithms are used to compress a long message to make it suitable for public key cryptography.
