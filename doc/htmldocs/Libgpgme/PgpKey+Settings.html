PgpKey.Settings Class

A class that contains general settings used during key edit operations.

public class PgpKey.Settings


Remarks

At the moment the user can specify a passphrase or passphrase callback method that is used by various key edit operations.

Note:

This DOES NOT work if GPG is configured to use an agent (or pin-entry software) to acquire the user's passphrase. It is highly recommended to use GPG with a passphrase agent.
