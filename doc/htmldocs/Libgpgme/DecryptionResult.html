Recipients Property

public Recipient Recipients { get; }

Recipients to which a message was encrypted.

Value

A Recipient object. null if the message was not encrypted.

Remarks

This is a linked list of recipients to which a message was encrypted. You can enumerate the particular recipients through the Recipient.Next property or by using the Recipient.GetEnumerator() method.
