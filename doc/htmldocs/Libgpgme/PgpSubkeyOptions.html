ExpirationDate Property

The expiration date of the new subkey.

Syntax

public DateTime ExpirationDate { get; set; }

Value

A DateTime.

Remarks

Note:

Depending on your timezone you might see the date 1970-01-01 UTC. This special date is used to mark a key to never expire. You can unset the expiration date with the PgpSubkeyOptions.MakeInfinitely() method.
