public DateTime ExpirationDate { set; get; }

⊟ Value

A DateTime.

⊟ Remarks

Note:

Depending on your timezone you might see the date 1970-01-01 UTC. This special date is used to mark a key to never expire. You can unset the expiration date with the PgpExpirationOptions.MakeInfinitely() method.
