DataEncoding Enum

Specifies the encoding of a GpgmeData object.

public enum DataEncoding


Remarks

This type specifies the encoding of a Libgpgme.GpgmeData object. This encoding is useful to give the backend a hint on the type of data.
Members
Member name Description
Armor This specifies that the data is encoded in an armored form as used by OpenPGP and PEM.
Base64 This specifies that the data is encoded using the Base-64 encoding scheme as used by MIME and other protocols.
Binary This specifies that the data is encoding in binary form; i.e. there is no special encoding.
None This specifies that the encoding is not known. This is the default for a new data object. The backend will try its best to detect the encoding automatically.
