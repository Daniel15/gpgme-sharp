Recipient Class

Information about a recipient of an encrypted text.

public class Recipient : IEnumerable<Recipient>

⊟ Remarks

This is an object used to store information about the recipients of an encrypted text which is decrypted in a decryption operation.

You can enumerate the particular recipients through the Recipient.Next property or by using the Recipient.GetEnumerator() method.
