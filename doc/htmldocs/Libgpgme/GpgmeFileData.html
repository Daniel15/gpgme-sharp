GpgmeFileData Class

File based data buffer.

public class GpgmeFileData : GpgmeStreamData

⊟ Remarks

File based data objects operate on streams. Only a small amount of data is stored in core at any time, so the size of the data objects is not limited by GPGME.
Note:

The current .NET implementation of the GpgmeFileData class is inherited from the GpgmeStreamData class. All data that will be read from - or written to the file stream needs to be passed from managed to unmanaged memory and vice versa. A more efficient implementation that directly passes a C file descriptor or stream to GPGME is planned for future releases.
