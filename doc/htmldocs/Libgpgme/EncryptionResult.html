gpgme-sharp : Libgpgme Namespace

EncryptionResult Class

A class that contains the status result of an encryption operation.

public class EncryptionResult


Remarks

During an encryption operation the Libgpgme.EncryptFlags specify whether trusted keys are required. Invalid recipients are keys that do not have a high enough validity in the keyring.

The Libgpgme.EncryptionResult object contains information about invalid recipients. In case of one ore more invalid recipients the plaintext might be encrypted for all valid recipients and returned (if this happens depends on the crypto engine).
