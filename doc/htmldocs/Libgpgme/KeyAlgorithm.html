gpgme-sharp : Libgpgme Namespace

KeyAlgorithm Enum

Specifies the set of all public key algorithms that are supported by GPGME.

public enum KeyAlgorithm


Remarks

Public key algorithms are used for encryption, decryption, signing and verification of signatures.
Members
Member name Description
DSA This value indicates DSA, the Digital Signature Algorithm.
ELG This value indicates ElGamal.
ELG_E This value also indicates ElGamal and is used specifically in GnuPG.
RSA This value indicates the RSA (Rivest, Shamir, Adleman) algorithm.
RSA_E This value indicates the RSA (Rivest, Shamir, Adleman) algorithm for encryption and decryption only.

Note:

Deprecated. Use KeyAlgorithm.RSA and set the appropriate capabilities when creating new keys.

RSA_S This value indicates the RSA (Rivest, Shamir, Adleman) algorithm for signing and verification only.

Note:

Deprecated. Use KeyAlgorithm.RSA and set the appropriate capabilities when creating new keys.
