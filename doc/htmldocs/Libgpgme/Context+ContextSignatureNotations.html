Context.ContextSignatureNotations Class

A class used to list and modify signature notations data of a context.

Syntax

public class Context.ContextSignatureNotations : IEnumerable<SignatureNotation>

Remarks

By using the member method Context.ContextSignatureNotations.Add(string, string, SignatureNotationFlags), you can attach arbitrary notation data to a signature. This information is then available to the user when the signature is verified.

Add Method

Adds notation data to the context.

public void Add (string name, string value, SignatureNotationFlags flags)

Parameters

name
A string that contains the Notation name.
value
A string that contains the Notation value.
flags
Documentation for this section has not yet been entered.

Remarks

The function adds the notation data with the name name and the value value to the context

Subsequent signing operations will include this notation data, as well as any other notation data that was added since the creation of the context or the last Context.ContextSignatureNotations.Clear() operation.

The arguments name and value must be strings in human-readable form. The flag SignatureNotationFlags.HumanReadable is implied (non-human-readable notation data is currently not supported). The strings must be in UTF-8 encoding (will be automatically transformed when using string).

If name is null, then value should be a policy URL.

Clear Method

Removes the notation data from the context.

public void Clear ()

Remarks

The function removes the notation data from the context. Subsequent signing operations from this context will not include any notation data.
Note:

Every context starts with an empty notation data list.

Get Method

Receives the notation data from the context.

Syntax

public SignatureNotation Get ()

Returns

If notation data is available, a SignatureNotation object is returned. If there is currently no notation data added for this context, null is returned.

Remarks

The function returns a linked list of notation data objects that are contained in the context. You can get the next SignatureNotation through the SignatureNotation.Next property or enumerate all entries through the SignatureNotation.GetEnumerator() method.
