KeyStore Class

A class that represents the default key store.

public class KeyStore : IKeyGenerator, IKeyStore


Remarks

The most cryptographic operations require that recipients or signers are specified. This is always done by specifying the respective keys that should be used for the operation. A KeyStore that implements Libgpgme.IKeyStore provides all methods needed to import, export, delete and enumerate such keys. Furthermore if it implements Libgpgme.IKeyGenerator it can be used to generate new keys as well.

You can use the Context.KeyStore property to receive an engine-specific default key store object.

Export Method

public void Export (string pattern, GpgmeData keydata)

Extracts public keys and returns them in a user-supplied data buffer.

Parameters

pattern
A string containing an engine specific expression that is used to limit the list of returned keys.
keydata
A destination GpgmeData buffer.

Remarks

The function extracts public keys and returns them in the data buffer keydata. The output format of the key data returned is determined by the ASCII armor attribute set for the context (see Context.Armor).

If pattern is null, all available keys are returned. Otherwise, pattern contains an engine specific expression that is used to limit the list to all keys matching the pattern.

GetKeyList Method

public Key[] GetKeyList (string pattern, bool secretOnly)

Initiates a key listing operation.

Parameters

pattern
A string containing an engine specific expression that is used to limit the list of returned Key objects.
secretOnly
If true, the returning list is restricted to secret keys only.

Returns

An array of Key objects. Depending on the engine PgpKeys or X509Keys are returned.

Remarks

If pattern is null, all available keys are returned. Otherwise, pattern contains an engine specific expression that is used to limit the list to all keys matching the pattern.

Note:

The total length of the pattern is restricted to an engine-specific maximum (a couple of hundred characters are usually accepted). The pattern should be used to restrict the search to a certain common name or user, not to list many specific keys at once by listing their fingerprints or key IDs.

If secretOnly is not false, the list is restricted to secret keys only.

The KeyStore.Context (context to which the KeyStore object is attached to) will be busy until all keys are received.

Required Assembly Versions:

  • gpgme-sharp 1.0.0.0

Import Method

public ImportResult Import (GpgmeData keydata)

Adds the keys in the user-supplied data buffer to the key ring.

Parameters

keydata
A GpgmeData object that contains the keys.

Returns

An ImportResult object.

Remarks

The function adds the keys in the data buffer keydata to the key ring of the crypto engine used by KeyStore.Context. The format of keydata can be ASCII armored, for example, but the details are specific to the crypto engine.
