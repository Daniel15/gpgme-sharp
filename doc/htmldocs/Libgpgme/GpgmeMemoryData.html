GpgmeMemoryData Class

Memory based data buffer.

public sealed class GpgmeMemoryData : GpgmeData

⊟ Remarks

Memory based data objects store all data in allocated memory. This is convenient, but only practical for an amount of data that is a fraction of the available physical memory. The data has to be copied from its source and to its destination, which can often be avoided by using one of the other data objects.

⊟ GpgmeMemoryData Constructor

public GpgmeMemoryData (string filename)

⊟ Parameters

filename
A string containing the requested filename.

⊟ Remarks

Creates a new GpgmeData object and fills it with the content of the file filename.

The whole file is read in at initialization time and the file is not used anymore after that. This is the only mode supported currently. Later, there might an option causing all reads to be delayed until the data is needed, but this is not yet implemented.
