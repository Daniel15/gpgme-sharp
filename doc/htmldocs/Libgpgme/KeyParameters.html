⊟ Remarks

Note:

Depending on your timezone you might see the date 1970-01-01 UTC. This special date is used to mark a key to never expire. You can unset the expiration date with the KeyParameters.MakeInfinitely() method.

⊟ PubkeyAlgorithm Property

The public key algorithm.

public KeyAlgorithm PubkeyAlgorithm { set; get; }

⊟ Value

A Libgpgme.KeyAlgorithm.

⊟ Remarks

The primary key algorithm MUST have the AlgorithmCapability.CanSign capability. The AlgorithmCapability.CanCert capability will be added automatically. You can modify the capabilities with the KeyParameters.PubkeyUsage property.

Note:

The following public key algorithms are valid for the primary key when creating OpenPGP keys:

Algorithm Description
KeyAlgorithm.DSA DSA
KeyAlgorithm.RSA RSA (you need to add the AlgorithmCapability.CanSign capability)
