EmptyPassphraseException Class

Exception that will be raised if the user enters an empty passphrase.

Syntax

public class EmptyPassphraseException : BadPassphraseException

Remarks

When changing the passphrase of a PGP key you can specify, if empty passwords should be allowed by using the PgpPassphraseOptions.EmptyOkay field.
