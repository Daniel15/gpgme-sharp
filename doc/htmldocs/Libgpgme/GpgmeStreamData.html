GpgmeStreamData Class

Stream based data buffer.

public class GpgmeStreamData : GpgmeCbsData

⊟ Remarks

The GpgmeStreamData class is inherited from the GpgmeCbsData class.
Note:

Make sure that your supplied System.IO.Stream object has a working implementation of the System.IO.Stream.Position property, otherwise GPGME's write() callback will not work.

Note:

Make sure that you set a proper GpgmeStreamData.iostream property. Your inherited class must override the following properies:
NameMethod
CanReadSystem.IO.Stream.CanRead
CanWriteSystem.IO.Stream.CanWrite
CanSeekSystem.IO.Stream.CanSeek
CanReleaseGpgmeCbsData.CanRelease

The GPGME backend will do callbacks only for methods for which their Can* properties returned true at initialisation time.
