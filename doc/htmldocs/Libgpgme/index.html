GpgME Class Library

Libgpgme Namespace

Namespace

This class library uses P/Invoke to call the Libgpgme API functions and wraps them into .Net objects. The wrapper classes are written in C#.

Currently supported systems (tested):

Windows XP/Vista (32bit)GPG4WIN version 1.1.4
Ubuntu Linux (32bit/64bit)libgpgme 1.2.0
