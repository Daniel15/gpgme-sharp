SelectedSignatures Field

The (key-) signatures that shall be revoked.

Syntax

public int[] SelectedSignatures

Remarks

Note:

You need to specify an (int) array containing the numbers of all (key-) signatures that shall be revoked. If you enumerate the list of (key-) signatures (UserId.Signatures property) to count and identify the correct numbers - make sure that you DON'T count "revocation" signatures/certificates. You can check this with the KeySignature.Revoked property for each (key-) signature in the list.

Example:
Example
# gpg --edit-key 321FEDCBA
Command > check
uid  Mallory <mallory@home.internal>
rev!        ABCDEF123 2009-08-01  Alice <alice@home.internal>
rev!        ABCDEF123 2009-08-02  Alice <alice@home.internal>
sig!3        321FEDCBA 2009-08-01  [self-signature]
sig!        ABCDEF123 2009-08-01  Alice <alice@home.internal>
There are four entries in this example list of (key-) signatures. The first two are "revocation signatures". That means the self-signature starts with number "1" and Alice's (key-)signature follows with number "2". (In this example, Alice created two revocation signatures/certificates. She created the first one on August 1st and the second one on August 2nd.)
