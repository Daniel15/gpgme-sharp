PassphraseDelegate Delegate

The type of method that usable as passphrase callback function.

public delegate PassphraseResult PassphraseDelegate (Context ctx, PassphraseInfo info, ref char[] passphrase)


Parameters

ctx
The context that initiates the passphrase callback.
info
A PassphraseInfo object that gives an indication for which user ID the passphrase is required.
passphrase
The returning passphrase.

Value

Returns a Libgpgme.PassphraseResult value that indicates success or a user canceled action.

Remarks

The PassphraseInfo object will be null in the case of symmetric encryption.

Note:

Make sure that you overwrite (wipe) the user's passphrase in memory. You should not use the string.ToArray() method to convert a user supplied string if you do not hold any further reference to it. The callback will NOT wipe the char array after usage.
