AmbiguousKeyException Class

Exception that will be raised if a key name or key id is ambiguous.

Syntax

public class AmbiguousKeyException : GpgmeException

Remarks

This Exception will be raised when trying to delete or searching for a particular key and the supplied name or id is ambiguous in the KeyStore (See KeyStore.GetKey(string, bool) and KeyStore.DeleteKey(Key, bool)).
