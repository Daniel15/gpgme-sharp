KeySignature Class

A class that contains information about a key signature.

public class KeySignature : IEnumerable<KeySignature>

⊟ Remarks

Key signatures are one component of a Key object, and validate user IDs (UserId) on the key. You can enumerate all key signatures by using the KeySignature.Next property or the KeySignature.GetEnumerator() method.

Note:

The signatures on a key are only available if the key was retrieved via a listing operation with the KeylistMode.Signatures flag enabled (see Context.KeylistMode), because it can be expensive to retrieve all signatures of a key.

The signature notations on a key signature are only available if the key was retrieved via a listing operation with the KeylistMode.SignatureNotations flag enabled (see Context.KeylistMode), because it can be expensive to retrieve all signature notations.

⊟ Expires Property

The expiration timestamp of the key signature.

public DateTime Expires { get; }

⊟ Value

A DateTime containing the expiration timestamp of the key signature.

⊟ Remarks

Always check the KeySignature.IsInfinitely property in order to find out whether the key signature can expire. If the signature cannot expire, KeySignature.Expires contains an invalid value (Depending on your timezone you might see the date 1970-01-01 UTC).
