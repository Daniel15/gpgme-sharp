GpgmeCbsData Class

An abstract class for callback based data buffers.

Syntax

public abstract class GpgmeCbsData : GpgmeData

Remarks

If neither memory nor file based data objects are a good fit for your application, you can inherit from this class and implement the required callback functions by yourself.

CanRelease Property

Indicates if GPGME will callback if a user-implemented data object should be destroyed.

Syntax

public abstract bool CanRelease { get; }

Value

A bool. If true, the inherited class must implement (override) the GpgmeCbsData.ReleaseCB() method.

Remarks

Release functions are called by GPGME if it wants to destroy a user-implemented data object.

LastCallbackException Field

The last exception that occurred during a callback from the GPGME backend operation.

Syntax

public Exception LastCallbackException

Remarks

Exception must not be propagated to P/Invoke functions. Therefore any exception that occurs during a callback will be saved into this field.

ReadCB Method

This is the function which GPGME calls if it wants to read data from a user-implemented data object.

Syntax

protected virtual IntPtr ReadCB (IntPtr bufPtr, long size)

Parameters

bufPtr
A pointer to the system memory where the requested number of bytes must be written.
size
The requested number of bytes.

Returns

The number of bytes that have been written OR GpgmeData.ERROR on any error.

Remarks

It is only required for input data objects (i.e. System.IO.Stream.CanRead is true).
Note:

The System.Runtime.InteropServices.Marshal.Copy method is recommended to copy data from managed to unmanaged memory.

SeekCB Method

This is the function which GPGME calls if it wants to change the current read/write position in a user-implemented data object (stream).

Syntax

protected virtual long SeekCB (long offset, System.IO.SeekOrigin whence)

Parameters

offset
Specifies the byte offset relative to origin.
whence
A System.IO.SeekOrigin value indicating the reference point used to obtain the new position.

Returns

Returns the new position within the current stream OR GpgmeData.ERROR on any error..

Remarks

It is only required if System.IO.Stream.CanSeek is true.

WriteCB Method

This is the function which GPGME calls if it wants to write data to a user-implemented data object.

Syntax

protected virtual IntPtr WriteCB (IntPtr bufPtr, long size)

Parameters

bufPtr
A pointer to the system memory where the requested number of bytes can be read.
size
The requested number of bytes.

Returns

The number of bytes that have been read from the memory and written into the user-implemented stream OR GpgmeData.ERROR on any error.
